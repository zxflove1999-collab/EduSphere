
## 1\. 认证管理 (Authentication)

### 1.1 用户登录

#### 1.1.1 基本信息

  * [cite\_start]**请求路径：** `/login` [cite: 526]
  * [cite\_start]**请求方式：** `POST` [cite: 527]
  * [cite\_start]**接口描述：** 该接口用于系统所有角色（学生、教师、管理员）登录 [cite: 88, 528][cite\_start]。使用用户名（学号/工号）和密码登录 [cite: 90, 134]。

#### 1.1.2 请求参数

  * [cite\_start]**格式：** `application/json` [cite: 530]
  * **参数说明：**

| 参数名 | 类型 | 是否必填 | 备注 |
| :--- | :--- | :--- | :--- |
| `username` | String | 是 | [cite\_start]登录账号 (学号/教工号/管理员账号) [cite: 134] |
| `password` | String | 是 | [cite\_start]登录密码 (初始密码为身份证后6位) [cite: 90] |

  * [cite\_start]**请求数据样例：** [cite: 534]

<!-- end list -->

```json
{
    "username": "2021001001",
    "password": "123456"
}
```

#### 1.1.3 响应数据

  * [cite\_start]**参数格式：** `application/json` [cite: 536]
  * **参数说明：**

| 参数名 | 类型 | 备注 |
| :--- | :--- | :--- |
| `code` | Integer | 响应码, 1为成功, 0为失败 |
| `msg` | String | 响应消息 |
| `data` | Object | 响应数据 (登录成功的用户信息) |
| `data.token` | String | [cite\_start]JWT令牌，后续请求需携带在Header中 [cite: 542] |
| `data.user_id` | Long | [cite\_start]用户ID [cite: 134] |
| `data.username` | String | [cite\_start]用户名 [cite: 134] |
| `data.full_name` | String | [cite\_start]用户真实姓名 [cite: 134] |
| `data.roles` | Array | [cite\_start]用户的角色标识符 `role_key` 数组 (如: `['student']` 或 `['teacher', 'resource_admin']`) [cite: 134] |

  * **响应数据样例：**

<!-- end list -->

```json
{
    "code": 1,
    "msg": "success",
    "data": {
        "user_id": 1001,
        "username": "2021001001",
        "full_name": "A同学",
        "token": "eyJhbGciOiJIUzI1NiJ9.eyJpZCI6MTAwMSwidXNlcm5hbWUiOiIyMDIxMDAxMDAxIiwicm9sZXMiOlsic3R1ZGVudCJdLCJleHAiOjE2OTg3MDE3NjJ9.w06EkRXTep6...",
        "roles": ["student"]
    }
}
```

## 2\. 个人中心 (Profile)

### 2.1 获取当前登录用户信息

#### 2.1.1 基本信息

  * **请求路径：** `/profile`
  * **请求方式：** `GET`
  * [cite\_start]**接口描述：** 获取当前登录用户的详细信息，用于首页欢迎 [cite: 111] [cite\_start]和个人资料页面 [cite: 130]。

#### 2.1.2 请求参数

  * **格式：** Header
  * **参数说明：**

| 参数名 | 类型 | 是否必填 | 备注 |
| :--- | :--- | :--- | :--- |
| `token` | String | 是 | [cite\_start]用户登录时下发的JWT令牌 [cite: 542] |

#### 2.1.3 响应数据

  * **参数格式：** `application/json`
  * **参数说明：** 响应包含 `t_user` 的基础信息和对应角色（学生/教师）的详细资料。

| 参数名 | 类型 | 备注 |
| :--- | :--- | :--- |
| `code` | Integer | 响应码, 1为成功, 0为失败 |
| `msg` | String | 响应消息 |
| `data` | Object | 用户详细信息 |
| `data.user_id` | Long | [cite\_start]用户ID [cite: 134] |
| `data.username` | String | [cite\_start]登录账号 [cite: 134] |
| `data.full_name` | String | [cite\_start]真实姓名 [cite: 134] |
| `data.email` | String | [cite\_start]电子邮箱 [cite: 134] |
| `data.phone_number` | String | [cite\_start]手机号码 [cite: 134] |
| `data.avatar_url` | String | [cite\_start]头像URL [cite: 134] |
| `data.roles` | Array | 用户的角色对象 (来自 `t_role`) |
| `data.profile` | Object | 详细资料 (学生或教师, 内容二选一) |
| `data.profile.major` | String | (学生) [cite\_start]专业 [cite: 134] |
| `data.profile.class_name` | String | (学生) [cite\_start]班级 [cite: 134] |
| `data.profile.department` | String | (教师) [cite\_start]所属院系 [cite: 134] |
| `data.profile.title` | String | (教师) [cite\_start]职称 [cite: 134] |
| `...` | | (其他 `t_student_profile` 或 `t_teacher_profile` 字段) |

  * **响应数据样例 (学生)：**

<!-- end list -->

```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "user_id": 1001,
    "username": "2021001001",
    "full_name": "A同学",
    "email": "student_a@edu.cn",
    "phone_number": "13800001111",
    "avatar_url": "/default_avatar.png",
    "roles": [
      { "role_id": 1, "role_name": "学生", "role_key": "student" }
    ],
    "profile": {
      "student_id_number": "2021001001",
      "gender": 1,
      "major": "计算机科学与技术",
      "class_name": "计算机2101班",
      "grade": "2021级"
    }
  }
}
```

### 2.2 修改当前用户信息

#### 2.2.1 基本信息

  * **请求路径：** `/profile`
  * **请求方式：** `PUT`
  * [cite\_start]**接口描述：** 用户在个人资料页面修改自己的信息 [cite: 89]。可修改 `t_user` 表中的 `email`, `phone_number` 和 `t_student/teacher_profile` 中的信息。

#### 2.2.2 请求参数

  * **格式：** `application/json`
  * **参数说明：** 提交 `t_user` 和 `profile` 两个对象。

| 参数名 | 类型 | 是否必填 | 备注 |
| :--- | :--- | :--- | :--- |
| `basicInfo` | Object | 是 | `t_user` 表中的可修改信息 |
| `basicInfo.email` | String | 否 | [cite\_start]电子邮箱 [cite: 134] |
| `basicInfo.phone_number` | String | 否 | [cite\_start]手机号码 [cite: 134] |
| `profileInfo` | Object | 是 | `t_student_profile` 或 `t_teacher_profile` 中的可修改信息 |
| `profileInfo.major` | String | 否 | (学生) 专业 |
| `profileInfo.office_location` | String | 否 | (教师) [cite\_start]办公室地点 [cite: 134] |
| `...` | | | (其他可修改的 profile 字段) |

  * **请求数据样例 (教师)：**

<!-- end list -->

```json
{
    "basicInfo": {
        "email": "teacher_b@edu.cn",
        "phone_number": "13900002222"
    },
    "profileInfo": {
        "office_location": "教三楼 305",
        "research_area": "人工智能、数据挖掘"
    }
}
```

#### 2.2.3 响应数据

  * [cite\_start]**参数格式：** `application/json` [cite: 216]
  * [cite\_start]**响应数据样例：** [cite: 220]

<!-- end list -->

```json
{
    "code": 1,
    "msg": "success",
    "data": null
}
```
